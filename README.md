# wetvlo

An app for downloading videos from various websites.

## Installation

```bash
bun install
```

## Available Commands

* `bun run build` - Compiles the TypeScript code into JavaScript and generates type declaration files
* `bun run lint` - Runs Biome and dprint checks to identify code quality issues
* `bun run format` - Formats the code using Biome and dprint
* `bun run typecheck` - Runs TypeScript type checking

## Development

Your main code should reside in `src/`.
